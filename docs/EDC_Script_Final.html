Introduction

Cette étude porte sur le jeu de données Gym Members Exercise Dataset.
L’objectif est de suivre une démarche d’analyse structurée permettant :

  1. de préparer et nettoyer les données ;
  2. de décrire les caractéristiques de l’échantillon ;
  3. d’explorer la structure des variables quantitatives par ACP ;
  4. de réaliser une classification non supervisée par CAH ;
  5. de construire des modèles de classification pour prédire une classe de dépense calorique ;
  6. de construire des modèles de régression pour prédire directement Calories_Burned.

La stratégie retenue combine ainsi une approche descriptive, exploratoire, non supervisée et supervisée.

2.3. Choix des variables retenues

Le nettoyage ne se limite pas à supprimer des observations. Il consiste aussi à limiter les redondances entre variables.

Dans cette étude :

  • BMI est une variable dérivée de Weight et Height et peut donc introduire une redondance ;
  • Max_BPM et Resting_BPM semblent a priori moins informatifs que Avg_BPM pour expliquer la dépense calorique ;
  • les analyses principales sont donc réalisées sur un sous-ensemble de variables quantitatives jugées plus pertinentes.

2.5. Création de la variable cible de classification

Afin de réaliser des modèles de classification, on transforme la variable continue Calories_Burned en trois classes de niveau de dépense calorique : Faible, Modérée et Haute, à partir des terciles.

burned_cal <- quantile(
  gym$Calories_Burned,
  probs = c(0, 1/3, 2/3, 1),
  na.rm = TRUE
)

gym$Classe_Calories <- cut(
  gym$Calories_Burned,
  breaks = burned_cal,
  include.lowest = TRUE,
  labels = c("Faible", "Moderee", "Haute")
)

gym$Classe_Calories <- as.factor(gym$Classe_Calories)

4.3. Choix du nombre d’axes

Deux critères sont utilisés ici : le critère de Kaiser et le seuil de Saporta.

n <- nrow(gym_quanti)
p <- ncol(gym_quanti)

kaiser_n <- sum(vp > 1)
saporta_threshold <- 1 + 2 * sqrt((p - 1) / (n - 1))
saporta_n <- sum(vp > saporta_threshold)

Le critère de Kaiser retient les axes dont la valeur propre est supérieure à 1.
Le critère de Saporta est plus exigeant et permet parfois une sélection plus prudente des dimensions réellement informatives.
